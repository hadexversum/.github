# HaDeXversum

We produce software for analysis of HDX-MS data. 

Our contributions:
 - [resources](): curated list of interesting articles from the HDX-MS field.
 - [HRaDeX](https://github.com/hadexversum/HRaDeX): R package for in-depth analysis of the uptake kinetics.
   - [HRaDeX web server](https://hradex.mslab-ibb.pl/): web server for single-state analysis.
   - [compaHRaDeX web server](https://compahradex.mslab-ibb.pl/): web server for comparative analysis.
   - [publication](https://pubs.acs.org/doi/10.1021/acs.jproteome.4c00700): article describing project.
 - [HaDeX2](https://github.com/hadexversum/HaDeX2): R package for standard analysis and processing of HDX-MS data.
   - [HaDeX2 web server](https://hadex2.mslab-ibb.pl/): web server. 
   - [HaDeXGUI](https://github.com/hadexversum/HaDeXGUI): R package for local deployment of HaDeX web server.
 - [HaDeX](https://github.com/hadexversum/HaDeX): first version of HaDeX.
   - [HaDeX GUI](https://hadex.mslab-ibb.pl/): web server.
   - [publication](https://academic.oup.com/bioinformatics/article/36/16/4516/5862011): article describing project.

  Authors: [Weronika Puchała](https://github.com/werpuc), [Michał Burdukiewicz](https://github.com/michbur).
